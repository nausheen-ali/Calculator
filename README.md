# Calculator
## 💡 Features

- Responsive UI with modern design
- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- AC (All Clear) button to reset input
- DE (Delete) button to remove the last digit
- Evaluates real-time expressions

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
